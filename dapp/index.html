Maybe Swap MaybeSwap allows anyone to maybify their swaps.

By maybifying, you either get more output than a normal swap or get nothing.

Rather than performing an atomic swap in a single transaction. Swapping is seperated into 2 transactions. First transaction does swap from user's input token to MAYBE, burn them to register a maybification and request VRF. Second transaction resolves it. If successful, more MAYBE gets minted and gets swapped back to user's desired output token.
